В программе PostgreKurs (С# Win Forms (Используемая библиотека: Npgsql)):

1.
Реализован вход через логин и пароль. 
(Логин и пароль записаны в таблице tbl_users и воспринимаются программой через функцию из базы u_login())
Доступны следующие:
1. Логин: admin  Пароль: 123 
2. Логин: kirsanov  Пароль: 12345
3. Логин: user  Пароль: user

2.
Реализован выбор определённой таблицы и составление отчётов
Отчёты представляют из себя файлы csv (Excel), в которых будут хранится данные из таблиц
Составление осуществляется через функцию db_to_csv() из базы
ВАЖНО! Отчёты сохраняются на диск С:\ в папку "DataKeeper"! Для успешного сохранения необходимо создать данную папку в корне диска!

3.
Для каждой таблицы реализован свой интерфейс где можно добавлять, изменять или удалять данные
(Как и в случае с окном авторизации и отчётами данные возможности также реализовываются в программе с использованием функций из базы. st_delete(),st_update_a_p() и т. д.)
(Для успешного внесения изменений в таблицу "Авто" в поле "Персональный ID" необходимо вместо имени указывать целочисленный ID владельца автомобиля)

4.
